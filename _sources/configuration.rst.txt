Configuration
===========================

Syntool can be configured with advanced versions of configuration files that provide more flexibility in retrosynthesis
planning and training of retrosynthetic models.

Planning configuration
---------------------------
Planning configuration yaml file

.. code-block:: yaml

    General:
      num_cpus: 4
      num_gpus: 1
      targets_path:
      results_root:
    InputData:
      reaction_rules_path:
      building_blocks_path:
      standardize_building_blocks: true
    PolicyNetwork:
      weights_path:
      priority_rules_fraction: 0.5
      top_rules: 50
      rule_prob_threshold: 0.0
    ValueNetwork:
      weights_path:
    Tree:
      ucb_type: uct
      c_ucb: 0.1
      max_depth: 9
      max_iterations: 100
      max_time: 120
      max_tree_size: 100000
      evaluation_mode: gcn
      evaluation_agg: max
      backprop_type: muzero
      init_node_value: 0.0
      silent: true


Training configuration
---------------------------
Training configuration yaml file

.. code-block:: yaml

    General:
      num_cpus: 4
      num_gpus: 1
      results_root:
    InputData:
      building_blocks_path:
      policy_data_path:
      reaction_data_path:
      value_data_path:
    DataCleaning:
      clean_reactions: true
      standardize_building_blocks: true
    ReactionRules:
      min_popularity: 3
      keep_leaving_groups: true
      reaction_center_neighbors: true
      reaction_center_hybridization: true
      reaction_center_implicit_hydrogens: false
      reaction_center_ring_sizes: false
      environment_neighbors: true
      environment_hybridization: false
      environment_implicit_hydrogens: false
      environment_ring_sizes: false
    Tree:
      backprop_type: muzero
      c_ucb: 0.1
      evaluation_agg: max
      evaluation_mode: gcn
      init_node_value: 0.0
      max_depth: 6
      max_iterations: 100
      max_time: 600
      max_tree_size: 1000000
      ucb_type: uct
      silent: true
    SelfTuning:
      batch_size: 100
      num_simulations: 1
    PolicyNetwork:
      policy_type: ranking
      batch_size: 500
      dropout: 0.4
      learning_rate: 0.0005
      num_conv_layers: 5
      num_epoch: 100
      priority_rules_fraction: 0.5
      rule_prob_threshold: 0.0
      top_rules: 50
      vector_dim: 512
    ValueNetwork:
      batch_size: 2
      dropout: 0.4
      learning_rate: 0.0005
      num_conv_layers: 5
      num_epoch: 100
      vector_dim: 512
