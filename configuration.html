Configuration¶

Syntool can be configured with advanced versions of configuration files that provide more flexibility in retrosynthesis planning and training of retrosynthetic models.

Planning configuration¶

Planning configuration yaml file

General:
  num_cpus: 4
  num_gpus: 1
  targets_path:
  results_root:
InputData:
  reaction_rules_path:
  building_blocks_path:
  standardize_building_blocks: true
PolicyNetwork:
  weights_path:
  priority_rules_fraction: 0.5
  top_rules: 50
  rule_prob_threshold: 0.0
ValueNetwork:
  weights_path:
Tree:
  ucb_type: uct
  c_ucb: 0.1
  max_depth: 9
  max_iterations: 100
  max_time: 120
  max_tree_size: 100000
  evaluation_mode: gcn
  evaluation_agg: max
  backprop_type: muzero
  init_node_value: 0.0
  silent: true
